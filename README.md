# Employee Management System (Java)

Employee Management System is a Java console-based application that performs CRUD operations on employee data using Object-Oriented Programming concepts. It allows users to add, view, search, update, and delete employee records using an interactive menu-driven system.

## Features
- Add Employee
- View Employees
- Search Employee by ID
- Update Employee Details
- Delete Employee

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner (User Input)

## Project Structure
- Model: Employee class
- Service: Business logic (EmployeeService)
- Main: Program execution (Main class)

## How to Run
Run `Main.java` in any Java IDE like Eclipse or IntelliJ.
